World Wide Web

En informatca, la World wide Web(WWW) ó Red informatica mundial comúnmente conocida como la WEB, es un sistema de distribución de dpcumentos de hipertexto o hipermedios interconectados o ascesibles via internet. Con un navegador web, un usuario visualiza sitios web compuestos de páginas web que pueden contener texto, imágenes, videos u otros contenidos multimedia,y navegar atravez de esas páginas usando hiperenlaces.
